# Fragile Code Ripple Effect Example

This example demonstrates how tightly coupled code can become 'fragile'. A small change in a core class's logic (e.g., adding a discount to an Order) can cause unexpected failures or inconsistencies in dependent modules that make assumptions about the core class's internal structure or calculation methods.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
